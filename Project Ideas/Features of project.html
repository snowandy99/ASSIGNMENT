Features of this product


Main Features

The main features of this application would be that messages which contain sensitive information between buyers and sellers can be encrypted via the app and can only be decrypted by the one device that the buyer/seller has created their account on.

Also with the use of an escrow system, the money sent from buyer to seller will be kept in a middleman service where the app will hold the money until released by the buyer or will automatically be released after 25 days.

If the item has not yet been received by the buyer after 21 days, the buyer can dispute the order and an administrator will get both parties  to communicate with each other to decide on what happens. Tracking on items must be included to prevent scamming acts from buyers and sellers.


Login Menu and Main Menu

To begin the app, there will be a login menu where you can also register an account to begin buying or selling. However, Two-Factor Authentication must be applied upon account creation to ensure that the account is slightly more secure. Also, you will be asked to decrypt a PGP message which will have a code to login ( kind of like a verification code ).

Once the user has logged in, the main menu will contain:

  • A list of categories which group up certain items together. Example: dog kennel would be under the category for 'Pets'
  • A preview of the user's balance of deposited money whether it be normal currency such as AUD, USD, YEN etc. or crytocurrency
  • A search bar so that the user can search for a specific item
  • An icon which leads to account settings in order to change passwords or details of the account

There will be a feedback system which be an important factor on a seller's trust rating. Buyers and sellers can leave feedback for each other to allow others to get a reading of whether they provide a good service or are good customers etc.
